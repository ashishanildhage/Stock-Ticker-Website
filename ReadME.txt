Steps to follow to run the Ticker Website:
1. Open Windows Terminal
2. Go to the directory in which this folder(Wright_Research_Ticker_app) is stored using cd.
3. Type and run seperately line by line after making sure you have python installed in your computer
	pip install flask
	pip install yfinance
	python server.py
4. Server gets deployed. 
5. Now open any web browser like chrome and run https://127.0.0.1:5000/
6. Type ticker name as ITC.NS, RELIANCE.NS and so on with .NS to get Indian NSE Stocks and MSFT, AAPL to get US stocks without .NS
